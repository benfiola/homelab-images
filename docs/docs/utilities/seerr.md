# seerr

A Docker image that extends [Seerr](https://seerr.dev/) with user-level media management. Users can delete their own requests and remove requested media from Sonarr/Radarr without requiring admin permissions. Built identically to the [official Seerr Docker image](https://hub.docker.com/r/seerr/seerr) with a minimal patch for user-level deletion capabilities.

## Configuration

Refer to the [Seerr Docker Hub documentation](https://hub.docker.com/r/seerr/seerr) for all environment variables, volumes, and configuration options.
