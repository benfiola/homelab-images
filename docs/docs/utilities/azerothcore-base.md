# azerothcore-base

A Docker base image that compiles [AzerothCore](https://www.azerothcore.org/) via the [mod-playerbots fork](https://github.com/mod-playerbots/azerothcore-wotlk). It includes the [mod-playerbots](https://github.com/mod-playerbots/mod-playerbots) module. Used as a dependency image for the [azerothcore](./azerothcore.md) image.
